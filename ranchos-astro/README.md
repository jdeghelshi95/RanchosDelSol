# Ranchos del Sol — Website

Beach-themed website built with Astro 5 + React + Tailwind CSS 4.

## Quick Start

```bash
npm install && npm run dev
```

Visit http://localhost:4321 | Admin at /admin (default password: ranchos2024admin)

## Features
- Car Rentals with admin CRUD, photo upload, reservations
- Stripe payments + email confirmations
- Stays, Boat Tours, Contact pages
- Beach-themed responsive design

## Build
```bash
npm run build && node dist/server/entry.mjs
```
